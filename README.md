# Western Australia Zhejiang Association Inc. Website

Official website of the Western Australia Zhejiang Association Inc. (WAZAI).

## About

The Western Australia Zhejiang Association Inc. is a non-profit community organisation dedicated to strengthening connections between Western Australia and Zhejiang through cultural exchange, community engagement, business collaboration, and social activities.

This website serves as the official online platform for sharing association information, news, events, and community resources.

## Features

* Bilingual support (English & Chinese)
* Responsive design for desktop and mobile devices
* Modern and accessible user interface
* Community events and announcements
* Photo gallery
* Contact information
* SEO-friendly structure
* Fast performance and optimized user experience

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* next-intl
* Vercel

## Project Structure

```text
src/

```

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```

## Production Build

Build the application:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Deployment

This project is designed for deployment on Vercel.

Deployment workflow:

```text
Local Development
        ↓
      GitHub

      Vercel
```
